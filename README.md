# Youngil WebApps

영일 관련 웹 애플리케이션 모음입니다.

## 시작하기

로컬에서 `*.html` 파일을 브라우저로 열거나, 정적 파일 서버로 호스팅하면 됩니다.

## 문서

| 파일 | 용도 |
|------|------|
| [PROJECT_CONTEXT.md](./PROJECT_CONTEXT.md) | 프로젝트 배경·구조·규칙 |
| [WORK_LOG.md](./WORK_LOG.md) | 작업 이력 |
| [TODO.md](./TODO.md) | 할 일 |
| [BUGS.md](./BUGS.md) | 알려진 이슈 |
| [DECISIONS.md](./DECISIONS.md) | 설계·정책 결정 기록 |

## 환경 변수

시크릿은 `.env`에 두고, 저장소에는 커밋하지 마세요. 템플릿은 `.env` 파일의 주석을 참고합니다.
